Proceso de Cotización y Contratación

1. Contacto Inicial

El proceso comienza cuando el cliente se pone en contacto a través de nuestro formulario de contacto, correo electrónico o teléfono. En este primer contacto, recopilaremos información básica sobre el proyecto.

2. Evaluación y Cotización

Una vez recibida la información inicial, realizaremos un análisis detallado de las necesidades y enviaremos una cotización formal dentro de las 48 horas hábiles. La cotización incluirá:

  • Desglose detallado de servicios
  • Tiempo estimado de desarrollo
  • Costos asociados
  • Términos de pago

3. Acuerdo y Contrato

Al aceptar la cotización, se procederá a la firma de un contrato que especificará:

  • Alcance del proyecto
  • Cronograma de trabajo
  • Términos de pago
  • Derechos y responsabilidades
  • Política de cambios y modificaciones

4. Proceso de Pago

Nuestro esquema de pago estándar es:

  • 50% de anticipo al iniciar el proyecto
  • 50% al finalizar el proyecto

5. Garantías y Soporte

Todos nuestros servicios incluyen:

  • 1 mes de mantenimiento gratuito
  • Garantía de satisfacción
  • Soporte técnico durante el desarrollo
  • Capacitación básica para el manejo del sitio

6. Política de Modificaciones

Durante el desarrollo del proyecto:

  • Se permiten ajustes menores sin costo adicional
  • Cambios significativos serán cotizados por separado
  • Toda modificación debe ser solicitada por escrito

7. Política de Entrega del Código Fuente

Respecto a la entrega del código fuente de los proyectos:

  • Por defecto, el código fuente no se incluye en la entrega del proyecto
  • Esta política existe para proteger nuestra propiedad intelectual y mantener la seguridad de nuestros desarrollos
  • Si el cliente desea obtener el código fuente completo del proyecto, esto tiene un costo adicional de USD 60
  • La entrega del código fuente incluye:
    • Todos los archivos fuente del proyecto
    • Documentación básica del código
    • Instrucciones de implementación
  • La transferencia del código fuente requiere la firma de un acuerdo adicional

Para más información o consultas específicas sobre nuestros términos legales, no dude en contactarnos.
